# v2cc.jsのTUI化

現状ではv2cc.jsは、プロセスをソートして、見つかった順にメッセージを送るが、
それをTUI経由でユーザー操作で調整できるようにしたい。

画面は以下のような感じで、
Statusは、テキスト送信の有効＼無効を切り替える。
Volumeは、マイクからの音量をリアルタイム表示する。
Detected: は、認識中でまだ送信されていない文字を表示する。画面に入りきらない場合はずらして表示し、末尾が常にみえるようにする。
Send To: は、送信するプロセスの名前を表示する。

操作は、 Enterキーで On/Offを切り替える。
矢印キーでプロセスを切り替える。。と左で一つ前、下と右で一つ後に移動する。末尾になったら最初のにもどる。
候補の表示はしない。何度か動かしているあいだに目的のがみつかることを期待する。
送信先は、起動直後は 未設定　という状態で、どこにも送信しない。


```
+--------- V2CC ----------------------+
| Status: ON                          |
| Volume: *****                       |
| Detected: xxxxxxxxxxyyyyzzzz        |
| Send To: Obsidian                   |
|                                     |
+-------------------------------------+
```
